# Login App

A Next.js application with a login page built using shadcn/ui components, matching the design from [shadcn login-03](https://ui.shadcn.com/view/new-york-v4/login-03).

## Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

The login page is available at `/login` (or automatically redirects from the home page).

## Tech Stack

- Next.js 14
- TypeScript
- Tailwind CSS
- shadcn/ui components

## Features

- Login page with Apple and Google OAuth buttons
- Email and password login form
- Responsive design
- Built with shadcn/ui components
